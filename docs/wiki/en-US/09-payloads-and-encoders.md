# Payloads and Encoders

**Language:** English (en-US). **pt-BR:** [../pt-BR/09-payloads-e-encoders.md](../pt-BR/09-payloads-e-encoders.md)

## Payloads

There are **32** payload modules spanning **x86**, **x64**, **armle**, **mipsbe**, **mipsle**, plus **`cmd`**, **`perl`**, **`php`**, and **`python`** categories for embedding into exploits or manual testing.

## Usage example

```text
RouterXPL-Forge > use payloads/cmd/bind_tcp
RouterXPL-Forge (bind_tcp) > show options
RouterXPL-Forge (bind_tcp) > run
```

## Encoders

**13** encoder modules provide **Python**, **PHP**, and **Perl** transformations (including **base64** and **hex** variants) to obfuscate or transport payload bytes.
